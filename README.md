# Snake Terminal Game in Go

A classic Snake game implemented in Go for terminal play.

## Features

- Simple controls
- High score tracking

## Requirements

- Go 1.18 or higher
## Installation
```bash
    > git clone https://github.com/0xjah/SnakeT.git
    > cd SnakT
    > go run main.go
```
